# UI Clock Changes - User Guide

## What's New

The Pill Assistant panel now includes visual clock displays and sorting options to help you better manage your medications.

## Clock Visualizations

### What They Show
Two circular clocks appear at the top of the Medications tab:
- **12-Hour Clock** (left): Shows times in traditional 12-hour format
- **24-Hour Clock** (right): Shows times in 24-hour (military) format

### How to Read the Clocks
1. Each clock has hour markers around the edge (1-12 or 0-23)
2. Wedges point to the exact time when each medication was taken
3. Multiple wedges at the same time stack together
4. The clocks update automatically when you take, skip, or snooze medications

- Clocks display side by side
- Full 300x300 pixel size
- Sorting buttons in a row

- Clocks stack vertically

## Dark Mode Support

All clock elements automatically adapt when you switch themes:
- Clock outlines adjust color
- Hour markers change shade
- Labels become more visible
- Sort buttons match theme

## Tips

1. **Check the clocks first** - Get a quick overview before scrolling through cards
2. **Use "Next Up" sorting** - See upcoming doses without searching
3. **Watch for red wedges** - Easily spot skipped medications
4. **Compare clock views** - Some people prefer 12-hour, others 24-hour format

## Troubleshooting

**Clocks appear empty?**
- Clocks only show medications that have been taken/skipped
- New medications won't appear until first action is recorded

**Wedges overlap?**
- Normal when multiple medications are taken at similar times
- Each wedge is 5 degrees wide for visibility

**Sort not working?**
- Make sure you're clicking the button, not just hovering
- Active sort button should be highlighted in blue

**Clocks not updating?**
- Clocks refresh automatically after medication actions
- If stuck, reload the page
